1.- border

These properties allow you to specify the style width and color of an element's border. The general syntax is the following:

border: [border-width || border-style || border-color | inherit] ;

Normally it is written as:

border: 1px #000000 solid;

2.- border-left, border-right, border-top, border-bottom

These properties create borders for different elements on the specified side.

border-left: 5px solid #0BA6D9;
border-right: 4px dotted #00DB43;
border-top: 3px double #00C2F0;
border-bottom: 2px dashed #36FA00;

3.- border-radius

This property makes borders with circular corners that doesn't apply when border-collapse is in collapse.

border-radius: 30px;

4.- background-image

it sets a one or more background images for an element 

background-image: url("kame.jpg");

5.- color

defines a color in either rgb or hexadecimal

color: #000000;

6.- font

this shorthand property sets all the different properties of an element's font, as with any shorthand property, any individual value that isn't specified is set to its default.

font: italic 1.2em "Fira Sans", serif;

7.- font-family

This property defines a prioritized list of one or more font family names or generic family names for the selected element.

font-family: 'Times New Roman', Times, serif;

8.- font-style, font-weight

  • font-style - this property sets a font to either itallic, oblique or normal face from its font-family.
  • font-weight - sets the weight (or boldness) of the font, its available weight depends on its font-family.

font-style: italic;
font-weight: bold;

9.- width, height

These properties set the size of an element through height and width, if max-width is used, the element is set to its maximum width. Their different values include:

  • auto - the default where the browser calculates height and width.
  • length - height/width in px, cm, etc.
  • % - height/width in percent of the containing block.
  • initial - height/width in default value.
  • inherit - height/width inherited from parent value.

height: 50px;
width: 100%;

10.- list-style

This property defines all the style parameters for a list.

  • Item 1
  • Item 2
  • Item 3

list-style: circle;

11.- margin

This property creates a margin in every single side, it is an abreviation to avoid using margin-(specific side) properties.

margin: 10px;

12.- padding

This property creates a filler area in each side to create space between borders and content. Each side can have its own specific padding size.

padding: 40px;

13.- overflow

This property specifies wether to clip the content or add scrollbars when a given content is too big to fit in a specific area. The different values fot this property are:

  • visible - default where overflow isn't clipped and content renders outside the box.
  • hidden - overflow is clipped and the rest od the content is invisible.
  • scroll - overflow is clipped and a scrollbar is added to see the rest of it.
  • auto - scroll but the bar is added only when necessary.

overflow: scroll;

14.- display

This property specifies the display behavior

(type of rendering)

of an element. It can have

different

values:
  • none
  • inline
  • block
  • inline-block
display: inline;

15.- position

The position property specifies the type of positioning an element will be given. It can have one of five different values:

  • static
  • relative
  • fixed
  • absolute
  • sticky

position: relative;

16.- filter

This property defines visual effects like blur or saturation to an element that often is <img>

filter: grayscale(30%);
